# Sistema de Notificações com Design Patterns

Projeto desenvolvido em Java com o objetivo de aplicar conceitos de Padrões de Projeto (Design Patterns).

## Padrões utilizados
- Strategy
- Factory
- Singleton

## Funcionalidades
- Envio de notificações por e-mail, SMS e WhatsApp
- Seleção dinâmica do tipo de notificação
- Configuração centralizada do sistema

## Objetivo
Demonstrar de forma prática a aplicação de padrões de projeto em um sistema simples e organizado.
